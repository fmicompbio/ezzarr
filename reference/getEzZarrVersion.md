# Get the version of ez-zarr python package

Get the version of ez-zarr python package

## Usage

``` r
getEzZarrVersion()
```

## Value

A character scalar indicating the version of ez-zarr installed in the
conda environment provided with the package.

## Author

Charlotte Soneson

## Examples

``` r
getEzZarrVersion()
#> [1] "0.4.2"
```
