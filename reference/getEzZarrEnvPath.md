# Get path to ez-zarr conda environment

Get path to ez-zarr conda environment

## Usage

``` r
getEzZarrEnvPath()
```

## Value

A character scalar indicating the path to the conda environment provided
by the package.

## Author

Charlotte Soneson

## Examples

``` r
getEzZarrEnvPath()
#> [1] "/Users/runner/Library/Caches/org.R-project.R/R/basilisk/1.22.0/ezzarr/0.1.9/ezzarr"
```
